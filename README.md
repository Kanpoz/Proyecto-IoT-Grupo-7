# Proyecto-IoT-Grupo-7

Toda la información relacionada con el desarrollo, funcionamiento y documentación técnica del proyecto de Estación IoT de Monitoreo de Calidad del Aire se encuentra disponible en el apartado **Wiki** de este mismo repositorio.  

En dicha sección se detallan los objetivos del proyecto, la arquitectura del sistema, los componentes utilizados, los estándares aplicados, la lógica de funcionamiento, así como las guías de implementación y operación. La Wiki actúa como la fuente principal de referencia para comprender el diseño, la justificación técnica y el proceso de desarrollo del sistema.
